About

This Reversi game is part of a class project for UC Irvine's Master in Human-Computer Interaction and Design program led by Professor Don J. Patterson .

The project uses a custom Node.js web server to manage application state and a Bootstrap 4 JavaScript web application for the user interface.

The aim of the project is to acquire a solid understanding of server-side and client-side web application programming as it relates to our Human-Computer Interaction and User Experience (UX) Design goals.
